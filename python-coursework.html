COURSEWORK IN PYTHON

Магазин продукции APPLE

Requirements for the website The website of the Apple online store is a website designed to demonstrate to visitors the entire range of a number of APPLE brand products. For each Apple product, the availability must be indicated (on the main page of the products in stock), price, photo and a brief description with characteristics. The site should contain the following sections: 1. Home page 2. Divisions 3. About Us page The site must contain an admin panel for the administrator, which represents the following functionality for them: 1.Adding new products to the catalog 2.Removing old products from the catalog 3. Viewing incoming orders The site data must be permanently stored in the database. The user interface of the website should be made in accordance with the latest UI/UX principles, providing ease of use and an attractive appearance for visitors.

CONCLUSION

The result of our work was the creation of a website, thanks to which all any buyer can get acquainted with our products and its full characteristics, they will also be able to quickly order the products they like. Administrators will be able to add products, delete missing ones, and see the list of orders. Contact customers. Work with product categories. Throughout the entire process of creating the course work, our team improved in writing code and studied a lot of useful information. Each participant contributed. We really worked as a team. Our work was hard, but quite interesting. We learned how to work better in a team, and improved our coding and documentation skills.
